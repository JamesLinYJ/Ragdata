# 烟草基础概览

本文档用于构建一个演示性质的“烟草知识库”。
内容适合用于术语检索、知识问答、页面联调与检索测试，不替代法规、医学或商业决策依据。

## 1. 烟草是什么

烟草通常指茄科烟草属植物及其加工产物。
在产业语境中，烟草既可以指烟叶原料，也可以指卷烟、雪茄、烟丝、加热不燃烧制品等终端产品体系。

## 2. 烟草知识库适合覆盖的内容

一个面向业务系统的烟草知识库，常见会包含以下几类信息：

- 基础术语：烟叶、烟丝、切丝宽度、含水率、配方、打叶复烤等
- 原料知识：烟草品种、产区、等级、成熟度、颜色、油分、结构
- 工艺知识：采收、烘烤、发酵、复烤、储存、切丝、卷接包
- 产品知识：卷烟类别、雪茄类别、规格、风格特征、香气特征
- 质量知识：感官评吸、质量缺陷、工艺偏差、稳定性控制
- 经营知识：品牌、渠道、陈列、消费者沟通、售后问答
- 合规知识：警示要求、营销限制、未成年人保护、区域政策

## 3. 知识库设计建议

如果要把烟草知识库用于智能问答或业务助手，建议把文档拆成以下结构：

1. 术语词条
2. 工艺流程说明
3. 产品介绍卡片
4. 常见问答
5. 制度或规范摘要

这样更利于检索、引用和后续前端页面展示。

## 4. 检索问答示例

以下是适合烟草知识库测试的示例问题：

- 什么是打叶复烤？
- 烟叶分级主要看哪些指标？
- 卷烟生产中含水率控制为什么重要？
- 烟草知识库里，工艺文件和制度文件应该如何分开管理？
